Plugin extends the functionality of WooCommerce to accept payments from credit/debit cards on your checkout page using PBS.

== Description ==

<h3>PBS Payment Gateway for WooCommerce</h3> makes your website ready to use PBS, to accept credit/debit cards on your e-commerce store checkout page in safe way. 

PBS is most widely used payment gateway to process payments on line and accepts Visa, MasterCard, Discover and other variants of cards.

<h3>WooCommerce 2.3.7 Compatible</h3>

= Features =
Few features of this plugin:

1. Accept Card right on your website.
2. No redirecting on other url.
3. Easy to install and configure
4. Option to configure success & failure message
5. Safe way to process credit cards and debit cards on WooCommerce using PBS
6. This plugin uses internal card processing, so faster and more relaible.

== Installation ==

Easy steps to install the plugin:

1. Upload 'pbs-payment-gateway-for-woocommerce' folder/directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce => Settings
4. On the "Settings" page, select "Payment Gateways" tab.
5. Under "Payment Gateways" you will find all the available gateways, select "PBS" option
6. On this page you wil find option to configure the plugin for use with WooCommerce
7. Enter the API details (MID,Gateway No,secret_key)
8. Configurable elements:

Title: This will appear on checkout page as name for this payment gateway

Description: This will appear on checkout page as description for this payment gateway

Merchant ID: This is the Login ID provided by PBS. (Note: this your User Merchant ID for Merchant Account)

Gateway No: This is Gateway No provided by PBS

secret key: This is Security Key provided by PBS

Transaction Success Message: This message will appear upon successful transaction. You can customize this message as per your need.

Transaction Failed Message: This message will appear when transaction will get failed/declined at payment gateway.

API Mode: This option sets the mode of API. Test/Sandbox Mode is when you need to test the gateway using some test transations. Live Mode to be used to process real transaction and card will actually get charged for this.

For Test Api Mode you can use publicly used

MID:20011
GatewayNo:20011004
Key:987654321

For Live Mode you can use your PBS provide information.
